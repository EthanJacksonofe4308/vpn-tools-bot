# VPN Tools Bot 🤖

Telegram bot with IP tools, speed test, and VPN link manager.

## Features
- 🌍 IP Geolocation
- 📊 Speed Test
- 🏓 Ping Test
- 🔍 IP Type Analysis
- 🔗 VPN Link Manager

## Installation

```bash
git clone https://github.com/EthanJacksonofe4308/vpn-tools-bot.git
cd vpn-tools-bot
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python main.py
